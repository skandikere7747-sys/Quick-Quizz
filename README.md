# QuickQuiz Web Application

QuickQuiz is a simple web-based quiz platform built using PHP, MySQL, HTML, CSS, and JavaScript.  
It allows users to register, log in, take quizzes, view scores, and track completed quizzes.

---

## Features
- User registration and login
- Session-based authentication
- Multiple-choice quizzes
- Score calculation
- User profile with quiz completion count
- Clean and easy-to-use interface

---

## Technologies Used
- PHP
- MySQL
- HTML / CSS
- JavaScript
- Apache (via XAMPP or online hosting)

---

## Local Setup (XAMPP)

1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder to:

xampp/htdocs/quickquiz

4. Open phpMyAdmin
5. Create database `quickquiz`
6. Import `db.sql`
7. Open browser:

http://localhost/quickquiz


---

## Online Deployment (InfinityFree)

1. Create a free account at https://infinityfree.net
2. Create a hosting account
3. Upload project files to `htdocs`
4. Create a MySQL database
5. Import `db.sql` using phpMyAdmin
6. Update `config.php` with database credentials
7. Access the site via your InfinityFree domain

---

Sample quiz questions are included in `db.sql`.
